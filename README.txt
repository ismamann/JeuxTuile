Le projet a été fait sur ECLIPSE.

Pour démarrer le jeu depuis le terminal :

Veuillez vous placer dans le dossier src/

export CLASSPATH=$CLASSPATH:$PWD
javac src/vue/VueLanceur.java
java src.vue.VueLanceur


En cas d'erreur pour les images, dans le fichier VueCarcassonne.java ligne 519 veuillez remplacer le chemin existant par le bon chemin.
